# navigation-router

A router for any framework base on the _Navigation API_.

## Getting Started

- Get code

```bash
git clone https://github.com/skyclouds2001/navigation-router.git
```

- Install dependencies

```bash
pnpm install
```

- Run

```bash
pnpm dev
```

- Build

```bash
pnpm build
```

- Preview

```bash
pnpm preview
```
